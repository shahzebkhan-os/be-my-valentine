# Be My Valentine

A cute and interactive Valentine's Day webpage with animated GIFs, floating hearts, and a playful "No" button that moves away when clicked!

## Features

- Beautiful gradient background with floating hearts
- Animated GIFs for visual appeal
- Interactive "Yes" and "No" buttons
- After clicking "No" 5 times, the button starts moving away
- When clicking "Yes", fireworks appear with a success message
- Fully responsive design that works on both desktop and mobile devices
- Playful animations and transitions

## How It Works

1. The user is presented with a cute Valentine's message and two buttons: "Yes" and "No"
2. If the user clicks "No", a pleading message appears
3. After 5 "No" clicks, the "No" button starts moving to random positions when hovered over
4. When the user finally clicks "Yes", a celebration page appears with fireworks animation

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- JavaScript (for interactivity)

## Usage

Simply open the `index.html` file in any modern web browser to enjoy the experience!
